Data Science Internship Projects – ShadowFox
This repository contains the projects completed during my Data Science Internship at ShadowFox.
The projects focus on data analysis, visualization, and machine learning techniques using Python.
🔹 1. Library Overview Analysis
Description:
This project analyzes library data to understand book availability, usage patterns, and category distribution.
Objectives:
Analyze library dataset
Identify most borrowed books and categories
Visualize library statistics using graphs
Tools & Technologies:
Python
Pandas
Matplotlib / Seaborn
Jupyter Notebook
Outcome:
The analysis helps understand reading trends and library usage patterns.
🔹 2. Sentiment Analysis
Description:
Sentiment analysis project that classifies text data into positive, negative, or neutral sentiments.
Objectives:
Perform text preprocessing
Apply sentiment classification
Analyze opinions from text data
Techniques Used:
Natural Language Processing (NLP)
Text cleaning and tokenization
Machine learning models
Tools:
Python
NLTK / Scikit-learn
Pandas
Outcome:
The model predicts sentiment from textual data and helps analyze public opinions.
🔹 3. Electric Vehicle Data Analysis
Description:

Objectives:

Visualize data insights
Tools & Technologies:
Python
Pandas
Matplotlib / Seaborn
Outcome:
The project provides insights into electric vehicle adoption and market trends.
